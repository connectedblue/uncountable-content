<p>A beautiful sunny day, and headed over to Pete&#8217;s house with all the equipment to finally assemble the traffic light hardware for the escape room game.</p>



<p>I had picked up the wooden box yesterday from Adrian, so we were able to start placing the various components in the optimum places for airflow and stability.  We started with the light itself where a hole had already been pre-cut.  </p>



<p>Once that was securely in, we fixed the hub and the pig tail and drilled holes for the power cables.  The final piece was the most delicate &#8211; fixing the pico and relay board with spacer screws onto the back of the box.  The fittings were slightly too large but a drop of hot glue fixed that.<br><br>The stage was now set for the wiring itself.  Two lights and the buzzer were connected to independent relays to allow maximum control from the tablet.  The testing worked perfectly first time, leaving only the front cover to fix into place.</p>



<p>We were very pleased to get this assembled.  It&#8217;s been quite a few weeks changing and perfecting the design, not to say the struggles I&#8217;ve had with the USB control interface from the tablet.  </p>



<p>But we are now in the home straight.  A few more software changes for me, and then it&#8217;ll be ready to install at the Escape Room for the opening in a few weeks.</p>
