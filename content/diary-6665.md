<p>They broke into teams again and worked very well on the different tasks.  The code itself is written in a scratch-like app called Lego spike which works directly in the browser.  We had some difficulty pairing the bluetooth, so fell back to the usb cable.  This is OK for testing, but I&#8217;ll need to look into getting the pairing working properly for the challenges themselves.</p>



<p>While the robot framework was being constructed, another pair took each of the sensors, connected them to the controller and practised some basic code to get them working.</p>



<p>The other teams finished off all the obstacles, and by the end of the two hours, they had a working robot, some test code scripts and all the completed obstacles.  This puts them in a good position to start to complete some of the challenges next time.</p>
